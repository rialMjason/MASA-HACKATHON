🚀 Quick Start Guide

1 View the Dashboard

Click "View Dashboard" above to see the interactive Southeast Asia map with 11 country flags. No data is required to start exploring the interface.

2 Enter Risk Metrics

Visit the Data Input Tool and enter your risk values for each country:

  • Physical Risk (0-10): Climate hazards like flooding, storms, sea-level rise
  • Physical Risk %: Progress bar percentage (0-100%)
  • Transition Risk (0-10): Economic and structural transition risks
  • Transition Risk %: Progress bar percentage (0-100%)
  • Liability Risk (0-10): Legal and financial liability risks
  • Liability Risk %: Progress bar percentage (0-100%)

3 Update Dashboard

After entering your data, click "Update Dashboard". The data will be saved and automatically loaded when you open the dashboard.

4 Explore Countries

In the dashboard, click on any country flag to:

  • Zoom into that country on the map
  • View a detailed description
  • See risk metrics with visual progress bars
  • View states/provinces in that country

⚙️ Features

  • ✅ Interactive map with zoom functionality
  • ✅ Country flags as clickable markers
  • ✅ Three risk metric types with visual progress bars
  • ✅ Automatic color-coding based on risk levels
  • ✅ States/provinces listing for each country

  • ✅ Data persistence using browser storage
  • ✅ Easy-to-use data input interface

📊 Data Format

When you're ready with the exact numbers, use this JSON format:

{
  "code": "TH",
  "physicalRisk": 7.5,           // Numeric value (0-10 scale)
  "physicalRiskPercent": 75,     // Percentage for visual bar (0-100)
  "transitionRisk": 5.2,
  "transitionRiskPercent": 52,
  "liabilityRisk": 3.8,
  "liabilityRiskPercent": 38
}
